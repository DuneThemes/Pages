    This is cool

    Say you didn't want to include an image on one part of the page, but just wanted to have a chunk of text. How would that be done? Pretty simply, actually.

    It's as simple as adding "text" to the caption div. Doing that will remove the hover effects, the background and change the colour to black. Therefore creating a chunk of text, that conforms to the grid layout!

    Also, as well as all that, you can include headers in this section. We recommend using the h3 tag, as it scales nicely. ALSO, you can include links. They're styled nicely.
